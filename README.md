# Focus Tapper

Focus Tapper is a simple HTML5 timing game. The goal is to tap the shrinking zone at the right moment. Smaller zone = higher points. If you miss, the round ends.

## How to Play
- Press **Start** to begin.
- A colored zone appears inside the circle and starts shrinking.
- **Tap on the zone** to score points.
- Tapping closer to the center gives higher rewards.
- Missing the zone ends the game.
- High score is saved automatically.

## Features
- Accurate hit detection
- Smooth shrinking animation
- Normal and Hard modes
- Mouse and touch support
- LocalStorage high score

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)

## Run Locally
Just open `index.html` in your browser.
